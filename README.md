A WPF audio recording pad application for Windows.

## Overview

Paddy is a Windows desktop application built with WPF (.NET 8) that provides a recording pad interface for capturing and managing audio recordings.

## Requirements

- Windows 10/11
- [.NET 8.0 Runtime](https://dotnet.microsoft.com/download/dotnet/8.0)

## Build

```powershell
dotnet build Paddy.csproj
```
